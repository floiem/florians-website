Übersicht

Quiz Must have Features
Aus einer Menge an Fragen sollen in zufällig ausgewählte Reihenfolge fünf Fragen gestellt werden.
Es gibt immer drei Antworten zur Auswahl, aber nur eine ist korrekt.
Wenn eine Antwort ausgewählt wurde gibt es für die korrekte Antwort einen Punkt.
Für Falsche Antworten gibt es null Punkte. Es darf nur eine Antwort ausgewählt werden.
Über einen "nächste Frage" Button wird die nächste Frage mit erneut drei Antworten zur Auswahl angezeigt.
Gestartet wird das Spiel über einen Klick in den obigen Textbereiche.
Wenn eine Antwort ausgewählt wurde, darf keine weitere Antwort auswählbar sein.
Das Spiel wird über das erneute Laden der Seite gestartet.

Quiz Bonusfeatures
- Die Antworten stehen direkt auf den Button, anstatt im oberen Textfeld
- Es gibt eine Highscore
- Es gibt eine Highscore Liste
- In jeder Runde gibt es unterschiedliche Fragen 
- Mutliplayer Modus z.B. Player1 und Player2 können gegeneinander spielen
- Visuelle Effekte z.B. beim knacken der Highscore
- Das gesamte CSS ist im Stylesheet
- Der Start des Spiels geht über einen gesonderten "Start Game" Button
